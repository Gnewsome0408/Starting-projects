USing CSS to expand your webpage

CSS provides your HTML with the design elements that make it appealing to an audience. For example, colour and spacing can be applied through these CSS definitions.

Summary: CSS introduction

  1. CSS treats each HTML element as if it appears inside its own box and uses rules to indicate how that element should look
  2. Rules are made up of selectors (that specify the elements the rule applies to) and declarations (that indicate what these elements should look like.
  3. Different types of selectors allow you to target your rules at different elemetns.
  4. Declarations are made up of two parts: the properties of the elements that you want to change, and the values of those properties.
  5. CSS rules usually appear in a seperate document, although tehy mauy appear within an HTML page.

Summary: Using Colour in CSS

  1. Colour not only brings your site to life, but also helps convey the mood and evokes reactions.
  2. There are three ways to specify colours in CSS: RGB values, hex codes, and colour names.
  3. Colour pickers can help you find the colour you want.
  4. It is important to ensure that there is enough contrast between any text and the background colour.
  5. CSS3 has introduced an extra value for RGB colours to indicate opacity. Known as RGBA.
  6. CSS3 also allows you to specify colours as HSL values, with an optional opacity value know as HSLA.
