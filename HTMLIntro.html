HTML introduction

The following is the first HTML page created for the learning Journal. This will provide some introduction points to using this language.

Basic tags to keep mind

  1. Open with HTML tag.
  2. Head and title tags will add the tab title and meta data.
  3. Body tag encompasses all material in the site
  4. H tags make the header, there are 6 levels
  5. P tags represent each paragraph.
